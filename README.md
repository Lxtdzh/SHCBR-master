# SHCBR

> A Global Structural Hypergraph Convolutional Model for Bundle Recommendation
# Environment Requirement
The code has been tested running under Python 3.6.13. The required packages are as follows:

numpy == 1.22.3

torch == 1.11.0_cu113

torch_sparse == 0.6.13

scipy == 1.7.3

# Usage
 ```python
python3 main.py
 ```
